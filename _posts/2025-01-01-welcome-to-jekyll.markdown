# Welcome to My New CMS-Powered Site!

This is your first blog post created with Jekyll and managed through Decap CMS. 

## What You Can Do

- **Create Posts**: Use the admin interface at `/admin` to create new blog posts
- **Edit Content**: Modify existing posts and pages without touching code
- **Upload Images**: Add images to your posts through the CMS interface
- **Manage Settings**: Update site-wide settings through the CMS

## Getting Started

1. Visit the [admin panel](/admin) to start creating content
2. Log in with your GitHub username and personal access token
3. Start writing!

Check out the [Jekyll docs][jekyll-docs] for more info on how to get the most out of Jekyll.
